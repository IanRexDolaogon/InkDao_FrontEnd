How to Run Locally

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1. Install Dependencies
Open your terminal, navigate to this project folder (`InkDao_FrontEnd`), and run:
```bash
npm install

2. Start the Development Server
Once the packages are installed, boot up the local Vite server:

Bash
npm run dev

3. View the App
Open your browser and navigate to the local address provided in the terminal (usually http://localhost:5173).
